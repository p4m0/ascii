# Small ascii project

Requirements.txt: \
keyboard==0.13.5 \
numpy==2.2.2 \
opencv-python==4.11.0.86 \

How to run:
1. pip install -r requirements.txt
2. Check that the PATH_TO_IMAGE is set correctly. 
3. Check the IMAGE_SCALE_FACTOR. Warning: Even "small" images may end up very big in ascii.
4. Run ascii.py
